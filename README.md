# skill-box

个人常用 Agent Skills 与编码规则的备份仓库。

## 目录说明

- `rule.md` —— 通用编码行为准则：Python/pip 命令使用习惯、CSV 编码、脚本清理、测试脚本的失败请求记录规范等，供 AI 编程助手在各项目中遵循。
- `skill_product_selection/` —— 1688 自动化选品 skill。根据自然语言选品需求，自动完成 关键词 → 1688 采集 → 数据分析 → Excel/Markdown 报告 全流程。
- `grill-me/` —— 方案设计深度追问 skill。在方案还停留在构思阶段时，逐层拆解决策分支，通过连续提问帮助理清设计中的隐藏假设和未决权衡。
- `test-report-generator-V2/` —— 测试报告生成 skill。根据需求描述、测试结论、问题单等信息，生成可直接复制到邮件中的简洁测试报告 Word 文档。
